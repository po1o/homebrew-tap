# homebrew

Homebrew tap for `po1o` formulas.

## Formulas

### `prompto`

Installs `prompto` from the binary artifacts published in
[`po1o/prompto` GitHub Releases](https://github.com/po1o/prompto/releases).

The formula selects the matching release asset for the current platform:

- macOS `amd64`
- macOS `arm64`
- Linux `amd64`
- Linux `arm64`

## Usage

```bash
brew tap po1o/tap
brew install prompto
```
